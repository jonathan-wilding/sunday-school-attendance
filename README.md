# Sunday School Attendance Tracker

A web application for tracking class attendance, built out of a real 
need - I'm the Sunday School secretary for my ward and got tired of 
managing attendance on paper. This project is both a genuinely useful 
tool for my class and my primary backend learning project as I work 
through my Software Engineering degree at BYU-Idaho.

## Why this project

Most class projects are built around a made-up problem. This one isn't - 
I know the actual workflow because I do it every week: who takes 
attendance, what they need to record, and where the current process 
breaks down. That context shapes every design decision here.

## Status

Early development. Currently building out the basic structure before 
adding database, authentication, and reporting features.

## Planned features

- Member roster management
- Weekly attendance entry
- Attendance history and reporting by member/class
- Simple, clean interface usable by any teacher, not just me
